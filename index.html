NextAva Health Widget

Embeddable clinic availability widget

Widget Files

The widget is hosted at:

https://nextavahealth.github.io/nextava-widget/nextava-widget.js

View Live Demo View on GitHub

Installation

Add these 2 lines to any webpage:

<div id="nextava-widget"></div> <script src="https://nextavahealth.github.io/nextava-widget/nextava-widget.js" data-clinic-id="YOUR_CLINIC_ID"></script>

Parameters:

  • data-clinic-id - Required. Your unique clinic identifier
  • data-color="#hexcode" - Optional. Custom brand color
  • data-show-updated="true" - Optional. Show last updated timestamp
  • data-branding="hide" - Optional. Hide "Powered by NextAva" footer

Example: Custom Color + Timestamp

<div id="nextava-widget"></div> <script src="https://nextavahealth.github.io/nextava-widget/nextava-widget.js" data-clinic-id="NWB001" data-color="#2563eb" data-show-updated="true"></script>
